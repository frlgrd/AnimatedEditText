Download
--------
Via Gradle:
```groovy
compile 'com.frlgrd:animated-edit-text:0.3'
```
or Maven:
```xml
<dependency>
  <groupId>com.frlgrd</groupId>
  <artifactId>animated-edit-text</artifactId>
  <version>0.3</version>
  <type>pom</type>
</dependency>
```
